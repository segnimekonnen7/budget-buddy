API Endpoints

Core Endpoints:
  • POST /api/analyze - Single text analysis
  • POST /api/batch-analyze - Batch processing
  • GET /api/twitter/:handle - Twitter sentiment
  • GET /api/analytics - Sentiment statistics
  • POST /api/custom-lexicon - Custom dictionary
Response Format:
{
  "text": "Sample text",
  "sentiment": "positive",
  "score": 0.85,
  "confidence": 0.92,
  "keywords": ["love", "amazing"],
  "timestamp": "2024-01-15T10:30:00Z"
}
