WebSocket Events & API Endpoints

WebSocket Events:
  • join-room - Join collaboration room
  • send-message - Send real-time message
  • typing - Typing indicator
  • code-change - Real-time code updates
  • user-presence - Online status updates
REST API Endpoints:
  • POST /api/auth/login - User authentication
  • GET /api/rooms - Get available rooms
  • POST /api/rooms - Create new room
  • GET /api/messages/:roomId - Get room messages
  • GET /api/users/online - Get online users

Sample Implementation

// WebSocket connection
const socket = io('http://localhost:3001');

// Join room
socket.emit('join-room', { roomId: 'general', userId: 'user123' });

// Send message
socket.emit('send-message', {
    roomId: 'general',
    message: 'Hello team!',
    userId: 'user123'
});

// Listen for messages
socket.on('new-message', (data) => {
    console.log('New message:', data.message);
});

// Typing indicator
socket.emit('typing', { roomId: 'general', userId: 'user123' });
